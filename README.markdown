Chatmore web-based IRC client
=============================

Github repo: https://github.com/spoulson/Chatmore  
Shawn Poulson, http://explodingcoder.com

SYSTEM REQUIREMENTS
-------------------

### Server
 - PHP 5.2+.
 - Linux/Unix OS.  Windows is not supported.

### Client
 - Tested on browsers: Firefox 5+, Chrome 13+, IE9.
